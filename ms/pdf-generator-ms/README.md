## Микросервис "Генератор PDF"

### Описание
Отвечает за генерацию PDF файла из шаблонов. Шаблоны хранятся в репозитории http://git.gosuslugi.local/luxoft/epgu2-services-json 
с json услуг, и создаются аналитиками при создании услуги. 

Если дефолтный шаблон отсутствует в репозитории, то микросервис генерит PDF автоматически, последовательно добавляя заполненные в черновике поля.
Внешний вид каждого поля зависит от типа компонента и описывается в отдельном шаблоне в услуге "default".

Rest-спецификация: http://localhost:8094/api/v3/api-docs
Swagger-ui: http://localhost:8094/api/swagger-ui/index.html?configUrl=/api/v3/api-docs/swagger-config
